# OpenAI setup

Use OpenAI if you already have an OpenAI project or want to use Atto's default model setup.

## Create a key

1. Open the OpenAI dashboard.
2. Create or choose the project you want Atto to use.
3. Create an API key and copy it once. The dashboard will not show the full key again.

## Paste and test

Keep `gpt-4o-mini` selected unless you already know you want another model. It is the default
because it is a practical balance of quality, speed, and cost for Atto's resume and job-search
workflows.

If the connection test fails, the most common causes are:

- The key was copied with a missing or extra character.
- The key belongs to a different OpenAI project than expected.
- The selected model is unavailable to the project.
- Billing, usage limits, or quota need attention.

After the test succeeds, save and proceed.
